# Labs

这些 lab 是 mini-vLLM 的教学练习。推荐顺序:

1. `01-token-accounting.md`
2. `02-paged-kv.md`
3. `03-continuous-batching.md`
4. `04-preemption.md`
5. `05-prefix-cache.md`
6. `06-benchmarking.md`

每个 lab 都遵循同一个节奏:

- 先读少量代码。
- 跑一个确定性实验。
- 改一两个参数。
- 用自己的话写下观察结论。

建议把实验输出和结论放到 `docs/notes/` 或个人笔记里, 不必提交所有临时结果。
